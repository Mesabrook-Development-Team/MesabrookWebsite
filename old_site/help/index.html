Help and Support

If you are experiencing any technical problems while trying to play on the Mesabrook server, please refer to this page for information.

Getting the Mesabrook Modpack

If your computer is running Windows, you will need to use Mesabrook's official companion program (MesaSuite) to download our modpack.

Unable to connect to Mesabrook or keep getting disconnected while playing.

If you have difficulties joining the Mesabrook server, make sure that...

  • Your computer is connected to the Internet and is able to access Mesabrook.com
  • Your Internet connection can handle multiplayer gaming.
  • Your computer or network's firewall isn't configured to block gaming traffic or block port 25565.

Make sure all of the Mesabrook Modpack's resource packs are enabled prior to joining.

Crashing

If you are experiencing Minecraft game crashes or Java Virtual Machine (JVM) crashes, please try the following...

  • Allocate more RAM to Minecraft.
  • Closing unnecessary programs when playing Minecraft.
  • Updating Java to the latest version of Java 8.

When Minecraft crashes, it will usually generate a crash report text file stored inside your .minecraft folder. If you report a crash experienced while on Mesabrook to us, please include that crash report text file.

How to find your crash-reports folder (Minecraft Wiki)

Low Framerates

If you are experiencing low framerates while playing on Mesabrook, please try the following...

  • Lower your Render Distance.
  • Set your Graphics to Fast.
